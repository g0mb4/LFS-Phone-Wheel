#Live for Speed Phone Wheel v1.1
===============================

What is it:
-----------
 It's a software, that uses the accelerometer of your phone to drive the car. I've made it, and you can donwload and share it for free. Written in C#(MV C# 2010) & Java (Eclipse).
 
Requirements:
------------- 
Phone:
 - Android 2.1+
 - accelerometer
 - WiFi

PC:
 - .net 4.0
 - LAN (Router)
 - LFS (S1 or S2)
 - keyboard & mouse :)
